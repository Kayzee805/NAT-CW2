# PSO-ANN

Small project to try Particle Swarm Optimization to optimize an Artificial Neural Network.

## Install dependencies

```
pip install -r requirements.txt
```

## Run tests

```
py.test
```

## Run example with MNIST

```
python example_mnist.py
```
